HOW TO ADD YOUR OWN PHOTOS
===========================

1. Open the "images" folder.
2. You'll see placeholder pictures already there (soft colored boxes with
   labels) — these just keep the site looking right until you replace them.
3. To use your own photo, replace a placeholder with your photo and give
   it the EXACT same file name. For example:

   - Want a new hero slider photo?  -> save it as images/hero-1.jpg
   - Want a new gallery photo?      -> save it as images/gallery-3.jpg
   - Want a new big portrait photo? -> save it as images/portrait.jpg
   - Want a new "make a wish" background? -> save it as images/wish-bg.jpg

   (If your photo is a .png instead of .jpg, just update the file
   extension in Birthday.html to match — search for the file name and
   change ".jpg" to ".png" right next to it.)

4. Recommended sizes (not required, just for the sharpest look):
   - hero-1.jpg ... hero-5.jpg   -> roughly 1600x1000 (landscape)
   - gallery-1.jpg ... gallery-8.jpg -> roughly 900x900 (square)
   - portrait.jpg                -> roughly 1000x1300 (portrait)
   - wish-bg.jpg                 -> roughly 1800x900 (wide)

5. To edit the text (her name, the quotes, the message, the countdown
   dates), open Birthday.html in any text editor (Notepad, TextEdit,
   VS Code, etc.) — everything is in plain readable text near the top
   of the <body> section.

6. To run the site: just double-click Birthday.html and it opens in
   your browser. No install, no server needed.

That's it — have fun making it yours. 🎂
